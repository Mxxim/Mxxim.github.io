我的第一个React练手项目——画廊应用

发表于 2016-12-31   |   分类于 React   |     |   阅读次数

前言

 好久没碰过这个博客了，在2016年的最后一天留个足迹吧！自从到了浙大读研，就意一直跟着导师做区块链研究，不怎么做前端项目了，囧～～但是，我对前端领域的技术更新会一直关注！刚好最近要做一个项目的宣传页，第一版UI的实现基本全部动画都是CSS3实现，没什么JS代码，后来UI改版，苦逼的我又得推倒重来。从2015年RN开源以来，我就一直想找个时间学习一下React，但却一直拖着。。。刚好抓住这个机会学习一下React和Webpack，然后把这个改版的前端界面架构搭起来。在慕课网上看到这个项目的教程，感觉挺有意思的，就用ES6语法做了起来。

关于React与Webpack的使用方法这里不阐述，建议大家都看官网学习，如果英文看的吃力的话再查中文资料。

项目准备过程

安装全局依赖

  • 安装nvm

  • 安装nodeJS: （当前版本是v6.9.1）

  • 安装Yeoman：（当前版本v1.8.5）

  • 安装generator-react-webpack：（当前版本v3.3.4）Yeoman的ReactJS项目生成器。

  • Chrome浏览器安装React Developer Tools扩展程序，方便Chrome Developer tool调试，这个扩展程序会在developer tool上定义一个__REACT_DEVTOOLS_GLOBAL_HOOK__全局变量，然后React在初始化的时候就会和这个HOOK进行通信，从而实现页面的热刷新。
